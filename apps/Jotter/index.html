Jotter - Notes app

A modern, open-source note-taking app built with Jetpack Compose. Focus on speed, simplicity, and privacy.

GitHub Playstore (Coming Soon)
(01) FEATURES

Rich Notes

Material You theming with dynamic colors. Multiple view modes (List, Grid) and Haptic feedback.

MATERIAL 3 DYNAMIC

Secure

Lock individual notes with a PIN or pattern. Includes Trash & Archive to keep you organized.

LOCK TRASH

Offline First

All data is stored locally using Room Database. No cloud sync, no tracking. Your data is yours.

LOCAL DB BACKUP
(02) SPECIFICATIONS

Jotter is built for an offline-first experience with a beautiful, dynamic UI. Backup and restore your notes easily.
